# Aura Sentinel IA

Sistema de IA para emergencias - 5 modelos offline

## Estructura del Proyecto

- **agentcore**: Clasificación de emergencias
- **geoguard**: Mapas y puntos de interés
- **chatlite**: Chat y apoyo emocional
- **resourcehub**: Ficha médica y recordatorios
- **voicelite**: Reconocimiento y síntesis de voz
